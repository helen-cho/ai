● AGENTS.md 와 Agent Skill

AGENTS.md와 Agent Skill은 AI 코딩 에이전트(예: Claude Code, Cursor 등)가 프로젝트를 더 잘 이해하고 특정 작업을 전문적으로 수행할 수 있도록 돕는 설정 및 지식 확장 체계이다. 두 개념은 AI 에이전트에게 맥락을 전달하는 방식과 목적에서 명확한 차이가 있다.

구분

AGENT.md (에이전트 정의 문서)

Agent Skill (에이전트 스킬)

정의

프로젝트 단위의 전반적인 작업 규칙 문서

에이전트가 특정 작업을 수행할 수 있도록 부여된 기능적 도구

범위

특정 프로젝트에 종속적

여러 프로젝트에서 독립적으로 재사용 가능

형태

텍스트 파일 (.md)

skill.md 파일이 포함된 하나의 폴더(디렉터리) 구조

방식

프롬프트 입력 시 에이전트가 항상 기억하고 참조하는 운영규칙

에이전트가 작업 중 필요할 때마다 스스로 호출하여 사용


1. 프로젝트 개발 시나리오

우리 회사는 홈페이지를 제작해주는 회사이다. 홈페이지에 사용되는 기술스택과 디자인 라이브러리는 직원 모두가 통일하여 사용하고 작성할 홈페이지의 카테고리(예: 의료기관, 교육기간, 쇼핑몰)에 따라 다른 메뉴와 다른 디자인으로 페이지를 생성하는 프로젝트를 작성한다.


[c:]-[data]-[cursor] 폴더에서 agents-skills 폴더를 생성하여 AGENTS.md와 Agent Skill을 이용한 홈페이지를 작성한다.


2. 프로젝트 작업 규칙 문서(AGENTS.md) 작성

목표: `AGENTS.md` 파일을 생성해서 이 문서에 프로젝트에서 사용할 기술 스택과 규칙을 정의하는 지침서를 작성한다.

[공통 기술스택]

1. HTML, CSS, JavaScript

2. 'Tailwind' CDN, 'lucide' CDN, Tailwind CSS를 사용한 'shadcn/ui'와 같은 디자인

[공통 개발규칙]

1. 로컬에서 실행될 단일 페이지로 구성

2. 모바일과 데스크톱 화면을 모두 지원 가능한 반응형 필수


3. 병원 홈페이지 스킬(Agent Skill) 작성

/create-skill

1. Skill name: clinic-skill-builder

2. Description

- 사용자가 병원, 의원, 치과, 피부과, 한의원 등 의료 기관의 홈페이지 제작을 요청했을 때 작동하는 프로젝트 스킬이다.

- 의료 기관에 필요한 신뢰감 있고 깔끔한 UI 구조와 업종별 필수 메뉴를 자동으로 주입한다.

3. Trigger Keywords

- 병원, 의원, 메디컬, 치과, 피부과, 안과, 성형외과, 내과, 한의원, 진료, 예약

4. Skill Prompt (AI 행동 지침)

[디자인 요구사항]

1. 주 색상: 청결함과 신뢰감을 주는 스카이블루 또는 에메랄드그린을 기본으로 사용한다.

2. 배경 색상: 무조건 깨끗한 화이트 또는 아주 밝은 그레이를 채택한다.

[네비게이션 바]

- 좌측에는 로고와 홈페이지 제목을 중앙에는 [병원소개]-[의료진 소개]-[진료안내]-[오시는 길 / 예약], 우측에는 전화번호

[메인 콘텐츠]

1. 히어로 섹션: '환자를 최우선으로 생각 합니다'와 같은 신뢰감을 주는 슬로건과 '빠른 예약하기' 버튼을 배치한다.

2. 병원소개(병원소개 글, 병원사진)과 의료진 소개(사진, 의료진 이름, 졸업대학 전공분야)를 깔끔하게 배치한다.

3. 진료시간 안내: 평일, 토요일, 점심시간, 일요일/공휴일 휴무 여부를 명확하게 보여주는 표(Table) 형태의 UI를 제공한다.

4. 오시는 길: 주소, 전화번호, 주차 안내 문구와 함께 지도(Map) 영역을 더미(Dummy) 박스로 시각화한다.

5. 하단 푸터(Footer): 대표전화번호, 이메일 주소, 저작권(Copyright)을 더미 데이터로 출력

6. 아이콘은 'lucide' 라이브러리를 사용해 병원 홈페이지에 어울리는 그래픽 요소들로 배치한다.

7. 카드(박스)에 마우스를 올리면 살짝 떠오르는 부드러운 애니메이션 효과와 고급스러운 그림자를 추가한다.

'튼튼한 정형외과' 병원 홈페이지를 `clinic.html` 파일로 만들어줘.

4. 교육기관 홈페이지 스킬(Agent Skill) 작성

/create-skill

1. Skill name: education-skill-builder

2. Description

- 사용자가 학교, 학원, 교육 기관, 인터넷 강의, 클래스 등의 홈페이지 제작을 요청했을 때 작동하는 프로젝트 스킬이다.

- 교육 기관에 걸맞은 체계적인 커리큘럼 구조와 학습 욕구를 자극하는 UI를 자동으로 주입한다.

3. Trigger Keywords

- 학교, 대학교, 고등학교, 학원, 코딩학원, 영어학원, 교육, 커리큘럼, 강좌, 클래스, 수강

4. Skill Prompt (AI 행동 지침)

[디자인 요구사항]

1. 주 색상(Primary Color): 지적이고 신뢰감을 주는 딥네이비와 포인트를 줄 수 있는 웜옐로우를 조합한다.

2. 레이아웃: 정보가 많아 보이므로 그리드(Grid)와 카드(Card) 형태를 적극 활용하여 정돈된 느낌을 준다.

[네비게이션 바]

- 좌측에는 로고와 홈페이지 제목, 중앙에는 [학원 소개]-[강사 프로필]-[교육 과정(커리큘럼)]-[수강 후기 / 공지사항], 우측에는 '수강상담' 버튼

[메인 콘텐츠]

1. 히어로 섹션: '당신의 성장을 설계합니다.'와 같은 전문성을 주는 슬로건과 '수강 상담하기' 버튼 배치한다.

2. 강사진 소개 카드: 강사 사진(더미), 이름, 대표 약력을 깔끔하게 정렬한 3열 카드 레이아웃

3. 커리큘럼 타임라인: 수강생이 어떤 순서로 공부하는지 한눈에 볼 수 있는 타임라인(Timeline) 또는 스텝(Step)바 UI 제공

4. 수강 후기 회전(Rolling) 보드: 실제 수강생들의 생생한 한 줄 평과 별점이 들어간 리뷰섹션 배치.

5. 하단 푸터(Footer): 대표전화번호, 이메일 주소, 저작권(Copyright)을 더미 데이터로 출력

6. 아이콘은 'lucide' 라이브러리를 사용해 교육기관 홈페이지에 어울리는 그래픽 요소들로 배치한다.

7. 카드(박스)에 마우스를 올리면 살짝 떠오르는 부드러운 애니메이션 효과와 고급스러운 그림자를 추가한다.

'스마트' 영어학원 홈페이지를 `education.html` 파일로 만들어줘.


5. 쇼핑몰 홈페이지 스킬(Agent Skill) 작성

/create-skill

1. Skill name: shopping-skill-builder

2. Description

- 사용자가 쇼핑몰, 마켓, 커머스, 스토어 등 상품 판매 목적의 웹사이트 제작을 요청했을 때 작동하는 프로젝트 스킬이다.

- 상품이 돋보이는 그리드 레이아웃, 장바구니 UI, 직관적인 구매 전환(CTA) 버튼 구조를 자동으로 주입한다.

3. Trigger Keywords

- 쇼핑몰, 마켓, 스토어, 커머스, 상점, 쇼핑, 옷가게, 굿즈샵, 편집숍, 상품리스트, 장바구니

4. Skill Prompt (AI 행동 지침)

[디자인 요구사항]

1. 주 색상: 구매 버튼이나 할인율 등 시선을 끌어야 하는 포인트에는 활력 있는 오렌지 또는 세련된 블랙을 사용한다.

2. 배경 색상: 상품 이미지가 가장 돋보이도록 잡음이 없는 순수한 화이트 배경을 사용한다.

[네비게이션 바]

- 좌측에는 로고 , 우측에는 [SHOP(상품)]-[ABOUT]-[EVENT]-[장바구니]-[마이페이지] 아이콘 배열 구조를 상단 헤더에 반드시 포함한다.

[메인 콘텐츠]

1. 히어로 섹션: 비주얼 배너(Carousel:캐러셀): 화면 전체를 채우는 대형 기획전 배너 이미지(더미)와 '시즌 오프 최대 50% 세일' 같은 카피라이팅 배치

2. 상품 격자 리스트 (Product Grid): 모바일에서는 2열, 데스크톱에서는 4열로 정렬된 상품 카드 섹션 구성

- 상품 카드 내부 요소: 상품 이미지(더미), 브랜드명(작게), 상품명, 할인율(레드 컬러) 및 판매가, '장바구니 담기' 퀵 버튼을 반드시 포함

3. 하단 푸터(Footer): 쇼핑몰 필수 정보인 사업자 등록번호, 통신판매업 신고번호, 고객센터 운영시간(1:1 톡 문의 버튼)을 가상 데이터로 하단에 배치

4. 아이콘은 'lucide' 라이브러리를 사용해 인테리어업체 홈페이지에 어울리는 그래픽 요소들로 배치한다.

5. 카드(박스)에 마우스를 올리면 살짝 떠오르는 부드러운 애니메이션 효과와 고급스러운 그림자를 추가

'오버핏 스튜디어' 패션 쇼핑몰 홈페이지를 `shopping.html` 파일로 만들어줘.
