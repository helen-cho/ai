3. 카드(박스)에 마우스를 올리면 살짝 떠오르는 부드러운 애니메이션 효과와 고급스러운 그림자를 추가한다.

4. 아이콘은 'lucide  라이브러리를 사용해 게임 홈페이지에 어울리는 그래픽 요소들로 배치한다.


2. 사과(Apple) 게임

사과 게임 규칙을 검색하여 `apple_game.md` 마크다운 파일에 저장한다.

[애플게임 요구사항]

- @apple_game.md 게임 규칙 기획서를 참고해서 프로그램 페이지를 작성한 후 '애플 게임' 상자를 클릭하면 해당 페이지로 이동한다.

[애플게임 추가 요구사항]

1. 페이지 상단 왼쪽에 점수, 남은 시간, 선택 합, 상단 오른쪽에 게임시작 버튼, 게임종료 버튼을 상자 안에 예쁘게 한 줄로 배치하여 추가한다.

2. 사과 아이콘을 상자에 80%로 채우고 글자를 사과 아이콘 안에 출력한다.

3. 게임종료 버튼을 클릭한 경우 '게임을 종료하시겠습니까?' 모달창(헤더: 기본 버튼과 같은 색상 테마)을 띄워 질의 후 '에'인 경우 홈페이지로 이동한다.


3. 알카노이드(Arkanoid) 게임

알카노이드 게임 규칙을 검색하여 `arkanoid_game.md` 마크다운 파일에 저장한다.

[알카노이드게임 요구사항]

- @arkanoid_game.md 게임 규칙 기획서를 참고해서 프로그램 페이지를 작성한 후 '알카노이드 게임' 상자를 클릭하면 해당 페이지로 이동한다.

[알카노이드 추가 요구사항]

단조로운 벽돌 패턴을 깨고 내구도와 속성을 가진 특수 벽돌들을 추가한다.

1. 일반 벽돌: 노란색이나 초록색으로 1대 맞으면 깨짐.

2. 단단한 벽돌: 회색 벽돌로, 3번 맞아야 깨짐. 맞을 때마다 금이 가는 시각적 효과(Cracked effect)를 준다.

3. 폭발 벽돌: 빨간색 벽돌로, 파괴될 때 상하좌우 인접한 벽돌을 함께 파괴하는 범위 데미지를 준다.

4. 보이지 않는 벽돌: 공이 처음 부딪히기 전까지는 투명했다가, 맞으면 나타나면서 2번째 타격에 깨지게 한다.
