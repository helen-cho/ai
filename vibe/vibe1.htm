● 바이브(Vibe) 코딩

바이브 코딩(Vibe Coding)은 사람이 직접 소스 코드를 타이핑하는 대신, 생성형 AI(LLM)와 자연어로 대화하며 원하는 결과물(앱이나 웹사이트)의 '느낌(Vibe)'과 기능을 설명해 개발하는 새로운 소프트웨어 개발 방식이다.


1. 바이브 코딩 개발 도구

바이브 코딩 개발 도구들은 모두 AI 기반 코딩 환경을 구축하는 핵심 기술 및 소프트웨어들이지만, 구동 방식(에디터형, 터미널형)과 출시 주체에 따라 뚜렷한 차이점을 가지고 있다. 현재 가장 많이 활용되는 대표적인 바이브 코딩 개발 도구는 다음과 같다.


1) Cursor(커서): 가장 대중적으로 쓰이는 AI 에디터이다. 기존 VS Code 환경과 완벽히 호환되며, 파일 전체의 맥락을 이해하고 여러 파일에 걸쳐 코드를 자동 수정 및 리팩토링하는 능력이 뛰어나다.

2) Claude Code(클로드 코드): OpenAI 출신의 핵심 연구원들이 설립한 앤트로픽(Anthropic)에서 개발하였으며 개발자와 대화하며 코드베이스 전체를 분석하고 개선안을 제시하는 등 고도화된 AI 코딩 지원에 강하다.

3) 코덱스(Codex): 챗GPT(ChatGPT)로 잘 알려진 오픈AI(OpenAI)에서 개발하였으며 프로젝트 폴더 전체를 읽고, 스스로 테스트를 실행하고, 디버깅 후 깃허브(GitHub) 브랜치 생성 및 풀 리퀘스트(PR) 오픈까지 전 과정을 자율적으로 처리한다.

4) 안티그래비티(Antigravity): 구글에서 만들었으며 단순히 코드를 추천해 주는 비서 역할을 넘어, 사용자가 말 한마디만 하면 스스로 계획을 세우고 컴퓨터의 파일과 터미널을 직접 조작해 앱을 완성하는 '에이전트 퍼스트(Agent-first)' 환경을 제공한다.


2. 필요한 프로그램 설치

커서(Cursor)를 활용해 바이브(Vibe) 코딩을 할 때 반드시 설치해야할 프로그램들을 다음과 같다.

1) VS-Code기반으로 만들어진 에디터를 설치한다. 단순한 챗봇연동을 넘어 개발 환경 전체가 AI 워크플로우에 최적화된 기능들을 제공한다.

https://cursor.com/ko/download

2) AI가 생성한 자바스크립트 코드를 컴퓨터에서 실행하고, 필요한 개발도구와 패키지를 다운로드하여 구동하기 위해서 Node.js를 설치한다.

https://nodejs.org/ko/download
