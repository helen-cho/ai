1) 자기소개 홈페이지
- Light, Dark 모드를 클릭한 경우 '정말로 변경하실래요?' 모달창을 띄워 예인 경우에만 변경한다.
- 모달창 헤더의 배경색과 글자색은 기본 버튼 테마를 적용한다.

2) 과일가게 홈페이지
- 장바구니 버튼을 클릭하면 아래 요구사항을 구현하는 페이지를 생성후 연결해줘.
[요구사항]
1. 장바구니 목록을 출력한다.
2. 장바구니 아이템의 수량을 수정한다.
3. 장바구니 아이템을 삭제한다.

- 장바구니 페이지 삭제 버튼 클릭시 '정말로 삭제하시겠습니까?' 모달창(헤더: 기본 버튼의 색상테마 사용)을 띄워 예인 경우에만 삭제한다.

3) 인테리어 홈페이지
1. 시공사례 이미지를 검색하여 다운로드하지말고 링크로 출력한다.
2. 우리 홈페이지와 어울리는 이미지를 검색하여 다운로드 하지말고 히어로섹션 배경이미지에 링크로 출력한다.
3. 히어로 이미지의 불투명도(opacity)를 100%로 변경하고 히어로 섹션 글자 색상을 이미지와 잘 어울리도록 변경한다.
4. 히어로 이미지 5개를 검색하여 다운로드한 후 슬릭슬라이드로 출력한다.
5. 히어로 이미지 좌,우 이동 버튼을 사이즈를 키워 예쁜 커스텀 버튼으로 변경한다.

4) 게임 홈페이지
벽돌 이미지를 검색하고 붙여 놓은 후 
- 위 이미지를 참고해서 벽돌이미지를 바꿔줘.
- 벽돌이 출력되는 배경은 투명하게 바꿔줘.

-메인 콘텐츠을 3열로 변경하고 '테트리스 게임' 카드상자를 추가한다.
-테스트리 게임 규칙을 검색하여 `tetris_game.md` 마크다운 파일에 저장한다.
-tetris_game.md 파일을 참고하여 게임 프로그램을 작성하여 '테트리스 게임' 카드상자와 연결한다.


5) 데이터베이스
- 메모앱 메모 목록의 내용들은 2줄만 보이게하고 2줄이 넘을경우에는 생략기호를 출력한다.
- 메모 목록의 각 메모 정보에 수정, 삭제 아이콘을 작성일과 같은행 오른쪽에 추가한다.
- 수정 아이큰 클릭시 라지 사이즈 모달창을 띄워 내용과 태그를 수정하는 기능을 추가한다.
- 메모 수정 모달창에서 내용이나 태그가 수정되기 전에는 취소, 저장 버튼을 Disabled 상태로 유지한다.
- 모든 페이지 넓이는 메인 콘텐츠 넓이와 같게 설정한다.

6)산악 커뮤니티
목표: 게시글에 댓글 관리 페이지를 작성하고 게시글 상세 페이지에 출력한다.

[데이터베이스 요구사항]
1. Prisma와 SQLite을 연동하여 현재 데이터베이스에 'PostComments' 테이블 스키마를 정의한다.
2, 테이블 필드: id, postId(게시글 id), writer(작성자:nickname), content(내용) createdAt(작성일), updatedAt(수정일)

[댓글 목록 기능 및 UX/UI 요구사항]
1. 최근에 작성된 댓글순으로 출력하고 댓글이 많은 경우 무한 스크롤링 기능을 추가한다.
2. 상단 왼쪽에 작성자, 작성일, 오른쪽에는 로그인한 작성자가 쓴 댓글인 경우 '수정', '삭제' 링크 텍스트를 출력한다.
3. 댓글 내용은 3줄만 출력하고 3줄 이상인 경우 '자세히', '간략히' 링크 텍스트 출력하고 토글(Toggle) 기능을 추가한다.

[댓글 등록 기능 및 UX/UI 요구사항]
1. 로그인한 사용자용 댓글 입력창을 출력한다.
2. '취소', '등록' 버튼은 입력내용이 없는 경우에는 Disabled 상태를 유지한다.

[댓글 삭제 기능 및 UX/UI 요구사항]
- 댓글을 삭제를 하는 경우 '정말로 삭제하시겠습니까?' 모발창을 띄운 후 '예'인 경우에만 삭제한다.

[댓글 수정 기능 및 UX/UI 요구사항]
1. 내용을 수정할 수있는 입력창으로 변경한다.
2. '취소', '저장' 버튼은 수정되지 않은 경우에는 Disabled 상태를 유지한다.
3. 댓글을 수정하는 경우 '정말로 수정하시겠습니까?' 모발창을 띄운 후 '예'인 경우에만 수정한다.
