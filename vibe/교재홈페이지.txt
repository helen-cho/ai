목표: 바이브 코딩 학습을 위한 메인 페이지를 작성한다. 
[기술스택]: HTML5 + CSS + JavaScript

[페이지 레이아웃]
1. 네비게이션 바(Navigation Bar): 
- 좌측에 우리 홈페이지와 어울리는 로고 와 '바이브코딩' 제목
- 우측에는 아래 메뉴를 출력한다.
  1) '홈페이지 제작'
  2) '게임 페이지 제작'
  3) 'Agents와 Agent Skill'
  4) '데이터베이스'
  5) '산악회 커뮤니티'
2. 히어로 섹션(Hero Section): 큰 제목("바이브 코딩(Vibe Coding)")과 간단한 설명, 그리고 [포트폴리오] 버튼이 있는 상단 영역
3. 메인 콘텐츠(Main Contents): 일단 빈공간으로 남겨 놓는다.
4. 푸터(Footer) : 홈페이지를 간단하게 더미 데이터를 생성해서 소개.

[디자인 요구사항]
- 테두리는 투박한 검은색 선, 배경은 흰색, 글씨는 검은색 위주로만 아주 기본적으로만 처리한다. 세련된 디자인은 2단계에서 처리한다.

[디자인 기술스택]
- Tailwind CSS CDN, lucide CDN

[디자인 요구사항]
1. 모바일과 데스크톱 화면을 모두 지원하는 반응형 필수
2. @DESIGN.md (Starbucks) 디자인 마크다운 파일을 참고하여 테마를 설정한다.
3. Tailwind CSS를 조합해서 'shadcn/ui' 디자인이랑 똑같이 생긴 '버튼', '카드상자'로 깔끔하게 교체한다.
4. 아이콘은 'lucide' 라이브러리를 사용해 인테리어업체 홈페이지에 어울리는 그래픽 요소들로 배치한다.
