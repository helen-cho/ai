## 2. 타이포그래피 (Typography)

- **폰트 패밀리**: Pretendard (프리텐다드)
  - 제목(타이틀): Pretendard **Bold/ExtraBold** (두꺼운 고딕 계열)
  - 본문: Pretendard **Regular**
- **기준 본문 스펙** (슬라이드 8 명시 값):
  - 크기: **22pt**
  - 자간(letter-spacing): **0**
  - 행간(line-height): **1.7**
- **텍스트 컬러**:
  - 제목/강조: `#204F6E`
  - 본문: 다크 그레이~네이비 계열 (배경 흰색 위)
  - 서브타이틀/캡션: `#6A96AD` 계열의 연한 톤
- **타이포 규칙**: 폰트는 1종만 사용, 강조는 굵기(Bold) 차이로만 구분, 자간보다 행간을 넓혀 가독성 확보.

## 4. 컴포넌트 / 그래픽 요소 (Components)

### 4.1 번호 배지 (섹션 인덱스 박스)
- 표지 외 본문 슬라이드(2~7페이지) 좌상단에 **숫자 + 라운드 사각 배지** 형태 ("01"~"06")
- 배지 색: `#204F6E` 윤곽, 내부 흰색, 숫자는 네이비
- 배지 옆에 **가로 구분선**(두께 약 2.8pt, 색 `#204F6E`)이 제목까지 연결

### 4.2 제목 + 가로선 헤더 패턴
- 모든 본문 슬라이드 상단: `숫자 배지 — 가로선 — 큰 제목` 순서로 좌→우 배치
- 가로선 두께: **2.8125pt**, 색상 `#204F6E`

### 4.3 카드형 박스 (강조 섹션)
- 둥근 사각형 박스, 배경색은 `#EEEEEE` 또는 `#DCE2E8`를 교차 사용
- 박스 좌측에 **세로 바(|)** 형태의 포인트 라인 (`#204F6E`)
- 박스 안에 소제목(Bold) + 본문(Regular) 구조

### 4.4 텍스트 하이라이트(마커)
- 본문 중 핵심 문장에 `#EEEEEE` 배경의 형광펜 스타일 하이라이트 적용
- 강조 문구는 Bold 처리

### 4.5 번호 리스트 카드 (결론 슬라이드)
- 좌/우 2열 그리드, 각 항목은 둥근 사각 카드(`#EEEEEE`/`#DCE2E8` 교차)
- 카드 내부: **큰 번호(Bold, 네이비)** + 세로 구분선(`|`) + 설명 텍스트

### 4.6 세로 디바이더 라인 (다단 비교 레이아웃)
- 좌/우 컬럼 비교 구조에서 컬럼 상단 제목 양옆에 짧은 세로선 사용
- 선 두께: 2.8125pt, 색상 `#204F6E`

### 4.7 이미지 박스
- 사진은 사각형 그대로 삽입, 별도 테두리/그림자 없음, 콘텐츠 카드 내부에 자연스럽게 배치

## 5. 표지/마무리 슬라이드 전용 요소

- **로고 텍스트**: 좌상단 "MIRI COMPANY" (Bold, 네이비)
- **연도/페이지 표시**: 우상단 숫자 (Bold, 네이비)
- **검색창 스타일 강조 라벨**: 캡슐형(Pill) 박스, 좌측 `|` 바 + 텍스트 + 우측 원형 아이콘(검색/체크 아이콘), 배경 `#EEEEEE`, 아이콘 원은 `#6A96AD` 계열
- **메인 타이틀**: 초대형 Bold 타이틀, 두 가지 색 조합(네이비 `#204F6E` + 라이트 블루 `#6A96AD`)로 키워드 색상 분리
- **서브 설명 텍스트**: 중앙 정렬, 옅은 톤
- **하단 정보 박스**(마무리 슬라이드): 둥근 사각형, 배경 `#DCE2E8`, 폰트 스펙 안내 텍스트 포함

## 6. 디자인 원칙 요약

1. **색상 제한**: 메인 컬러 1개(네이비) + 보조 블루그레이 1~2단계, 총 2색 원칙
2. **폰트 제한**: 1종(Pretendard) + 굵기 차이로만 위계 구분
3. **여백/간격**: 자간은 0, 행간은 넓게(1.7) → 가독성 우선
4. **구조 일관성**: 모든 본문 슬라이드 = 번호 배지 + 가로선 + 제목 헤더 패턴 고정
5. **강조 방식**: 색상 강조보다 하이라이트 배경(#EEEEEE) + Bold 텍스트로 처리
6. **테두리 프레임**: 바깥 배경색 띠 + 흰색 액자 + 흰색 콘텐츠 카드의 3단 레이어 구조 고정
7. **푸터 통일**: 모든 슬라이드 하단에 동일한 옅은 블루그레이 바 + "BUSINESS TEMPLATE" 워터마크
