■ 회의명: 차세대 고객 경험(CX) 통합 플랫폼 구축 프로젝트 4차 정기 점검 회의
■ 일시: 2026년 6월 10일 (수) 오전 10:00 ~ 11:30
■ 참석자: 박민우 팀장(PMO), 김철수 대리(개발팀), 이영희 과장(마케팅팀), 최윤아 대리(디자인팀)

[회의 안건]
1. 각 부서별 상반기 개발 및 디자인 진척 상황 공유
2. UI/UX 화면 설계서 확정 및 프론트엔드 연동 일정 조율
3. 마케팅 자동화 솔루션 API 연동 테스트 리스크 점검

[회의 내용 및 부서별 공유 사항]

1. 디자인팀 (최윤아 대리)
- 모바일 앱 주요 화면(메인, 상품 상세, 결제 페이지) UI/UX 설계안 최종 확정 완료함.
- 디자인 에셋(아이콘, 이미지 소스) 개발팀 전달 준비 완료. 다만, 퍼블리싱 단계에서 피드백 수렴을 위한 2일 간의 버퍼 기간 필요 요구.
- 다음 주 화요일(6월 16일)까지 개발팀에 최종 Figma 가이드라인 링크 송부 예정.

2. 개발팀 (김철수 대리)
- 백엔드 핵심 DB 구축 및 회원가입/인증 모듈 개발은 당초 계획대로 90% 이상 완료됨.
- [긴급 리스크]: 타사 마케팅 자동화 솔루션의 신규 API 문서 업데이트 지연으로 인해 연동 테스트가 올스톱된 상황. 상대 측 담당자 회신이 일주일째 없는 상태임.
- 이로 인해 프론트엔드 화면 연동 일정이 최소 5영업일 이상 지연될 가능성 있음.

3. 마케팅팀 (이영희 과장)
- 오픈 기념 프로모션 이벤트 기획안 및 카피라이팅 초안 수립 완료.
- API 연동 지연으로 데이터 수집 테스트가 늦어지면 타깃 고객 분류 마케팅 세팅도 동반 지연 우려됨.
- 재경팀과 조율하여 이벤트 경품 예산 1,500만 원 승인 완료함.

[PMO 지시사항 및 결정사항 - 박민우 팀장]
- 개발팀의 API 연동 지연 리스크는 프로젝트 전체 타임라인에 치명적이므로 긴급 안건으로 격상함. 마케팅팀 이영희 과장이 외부 솔루션 업체 영업 담당자 라인을 통해 오늘 오후 내로 공식 핫라인을 개설하고 피드백을 받아낼 것.
- 디자인팀은 6월 16일 가이드라인 배포 후 개발팀 김철수 대리와 6월 18일까지 화면 싱크 맞추는 미팅을 별도로 가질 것.
- 다음 주 수요일(6월 17일) 5차 회의 때 API 이슈 해결 여부를 최우선으로 재점검하겠음.
